Library Management System

A desktop-based Library Management System developed using C#, .NET Framework, and SQL Server. The application helps manage library operations such as book records, student records, book issuing, and returns through a user-friendly interface.

Features
Add, update, and delete book records
Manage student information
Issue and return books
Search books and student records
Database integration with SQL Server
Simple and user-friendly interface
Technologies Used
C#
.NET Framework
Windows Forms
SQL Server
Project Objective

The objective of this project is to automate library management tasks, reduce manual work, and improve the efficiency of maintaining books and student records.
